# 中国城市轨道交通地铁碳滑板年度用量、供应商格局与行业趋势研究报告
## 1 研究背景与碳滑板核心价值界定

本章旨在系统梳理碳滑板在城市轨道交通，特别是地铁系统中的核心功能、技术标准、消耗特性与价值定位，为后续的用量估算、市场格局与趋势分析奠定坚实的理论基础。碳滑板虽为部件，却是列车动力传输的“咽喉要道”，其性能与消耗规律直接关系到运营安全、效率与成本。

### 1.1 碳滑板在城市轨道交通受流系统中的功能定位

碳滑板是城市轨道交通车辆受流系统的核心部件之一。在电力机车或电动车组中，受电弓或受流器通过升起与架空接触网或第三轨接触，从而为列车获取电能。碳滑板正是安装于受电弓或受流器上，**直接与外部电源导线发生接触并传导电流的继电器** [^1]。其核心功能是实现列车在静止或高速滑行状态下，与供电网络之间**连续、稳定、高效的电力传输**，将外部电能引入车辆内部，转化为牵引动力及车载系统用电[^2][^3][^4]。

这一功能定位决定了碳滑板在轨道交通系统中的关键性。美尔森等专业供应商将其产品描述为旨在助力电流传输实现稳定性和耐久性的解决方案[^2]。**没有碳滑板的顺畅工作，地铁列车就会失去动力源泉**，其性能的优劣直接影响到供电的稳定性、列车运行的可靠性乃至行车安全[^4]。因此，碳滑板是连接列车与电网、保障地铁系统“血脉”畅通的“关键之匙”[^4]。

### 1.2 碳滑板技术标准体系与核心性能要求

为确保碳滑板产品的可靠性、安全性与互换性，中国已建立了一套较为完善的国家标准体系，主要涵盖性能检测方法与产品技术规范两个层面。

**1. 性能检测方法标准：《GB/T 34572-2017 轨道交通 受流系统 受电弓碳滑板试验方法》**
该标准为强制性国家标准，于2018年4月1日实施，为碳滑板的本体性能检测提供了统一的技术依据[^5]。其核心在于规范了涵盖外观尺寸、物理机械性能、电气特性等在内的15项核心指标的试验方法[^5]。标准修改采用了国际电工委员会标准IEC 62499:2008，并进行了本土化改进，例如增加了气密性检测项目，修改了载流量试验的电流施加梯度等[^5]。该标准构建了从原材料到成品的完整质量验证流程，其核心试验项目可概括如下表所示：

| 试验类别 | 核心检测项目 | 关键指标/试验条件 | 标准依据 |
| :--- | :--- | :--- | :--- |
| **物理性能检测** | 外观尺寸检查 | 长度公差、表面平整度等 | GB/T 34572-2017 |
| | 碳条性能检验 | 电阻率 ≤ 15μΩ·m，洛氏硬度 HRB 85-95 | GB/T 34572-2017 |
| | 抗折强度测试 | ≥ 120MPa（三点弯曲试验） | GB/T 34572-2017 |
| **电气特性验证** | 载流量试验 | 模拟最高4000A持续30分钟运行，记录温升与接触电阻 | GB/T 34572-2017 |
| **机械耐久测试** | 磨耗性能试验 | 接触压力60N、速度100km/h下进行10万次循环，磨耗量≤原始厚度15% | GB/T 34572-2017 |

**2. 产品技术规范标准：《GB/T 38955-2020 城市轨道交通车辆用炭滑板》**
该标准为推荐性国家标准，于2020年12月1日实施，替代了2015年版本[^6]。它明确了适用于地铁、轻轨等城市轨道交通车辆受电弓的炭滑板产品的技术要求、检验规则、包装运输等规范[^6]。标准由大同新成新材料股份有限公司、冶金工业信息标准研究院、北京交通大学等多家单位联合起草，其技术内容与《GB/T 34572-2017》的试验方法相衔接，并与《JB/T 9582-2017 电力机车炭滑板》等标准共同构成了轨道交通受流系统的技术体系[^6]。

**这两项标准共同界定了碳滑板产品的技术边界与质量门槛**，为设备的设计制造、采购验收与质量管控提供了权威依据，确保了产品在复杂工况下的性能与可靠性。

### 1.3 碳滑板磨损机制、更换标准与消耗特性分析

碳滑板在工作过程中与接触网导线持续摩擦，属于典型的关键运行耗材。其消耗主要源于两种磨损形式的共同作用：**机械磨损和电气磨损**[^1]。机械磨损由碳滑板与接触网导线之间的物理摩擦与冲击造成；电气磨损则主要源于电流通过接触面时产生的电弧（拉弧）烧蚀[^1][^4]。地铁列车频繁的启停、加速与运行环境中的杂质（如灰尘、潮湿）会加剧这两种磨损[^4]。

基于安全与性能要求，行业形成了明确的更换标准。当碳滑板出现以下情况时必须更换：**磨损至规定的最小厚度值**（如南京地铁案例中为碳刷条最薄处小于4.5mm[^1]）、存在**横向贯通性裂纹或断裂**、纵向裂纹超限、滑板边缘缺损、滑板面因电弧灼伤形成凹凸不平、以及材质剥离或碳条松动等[^7]。这些标准通常依据《TB/T 3271 电力机车受电弓滑板》等行业规范制定，核心目的是保障结构安全与电流传输的可靠性[^7]。

从消耗规律来看，碳滑板的磨损是一个相对稳定但受多重因素影响的过程。以南京地铁的运营数据为例，其一号线在2008至2011年间，每年碳滑板更换数量稳定在80条左右，相当于每年更换一批，消耗率较为稳定[^1]。经测算，其碳滑板初始厚度22mm，按年均运营10万公里计算，**平均每万公里消耗约1.75mm**，低于业内同类产品平均水平[^1]。然而，消耗量也会因**车辆大修更换不同品牌（如国产）产品导致寿命差异**、**线路客流量及上线运营强度变化**等因素而发生显著波动[^1]。这揭示了碳滑板消耗不仅与运营里程线性相关，更与产品本身的质量（材料特性、摩擦学性能[^8]）、运营环境及维护策略紧密相连。

### 1.4 报告研究范围界定与碳滑板核心价值总结

基于上述分析，本报告将研究范围明确界定为：**中国城市轨道交通系统中的地铁领域（主要研究对象）**，其所使用的碳滑板产品需符合《GB/T 38955-2020》和《GB/T 34572-2017》等国家标准的技术要求。此界定排除了干线铁路、电力机车等其他场景，使后续的用量估算与市场分析更具针对性与准确性。

综上所述，碳滑板的核心价值体现在三个层面：
1.  **安全与运行保障价值**：作为受流系统的直接接触点，其性能是列车获得持续、稳定动力的根本前提，任何失效都可能引发供电中断甚至安全事故[^7][^4]。
2.  **全生命周期成本优化价值**：碳滑板是可预测的关键耗材，其标准化质量、可量化的磨损率及明确的更换标准，为运营单位制定科学的维护计划、精准控制备件库存与维修成本提供了基础[^1][^4]。
3.  **技术发展载体价值**：碳滑板的材料演进（如从纯碳到浸金属碳[^1]）、性能优化（如提升耐磨性与导电性[^8]）及智能监测技术的应用（如便携式磨耗测量仪[^4]），是推动轨道交通受流系统技术进步与运维模式现代化的重要缩影。

因此，深入研究其用量、供应链与趋势，对于把握轨道交通关键部件市场动态、提升运营经济性与安全性具有重要现实意义。

## 2 地铁碳滑板年度用量估算模型与结果分析

本章旨在构建并应用一个基于核心运营变量的碳滑板年度用量估算模型，以量化中国地铁领域的年度总需求量。模型将严格依据行业权威统计数据，并结合第一章已界定的碳滑板技术标准与磨损消耗特性来设定关键参数。本章将首先界定模型的核心变量与数据来源，然后构建估算公式并代入数据进行全国及重点城市层面的测算，最后分析不同运营强度与城市规模对用量差异的影响，并讨论模型的不确定性来源与适用范围，从而为后续市场格局分析提供量化基础。

### 2.1 估算模型核心变量界定与数据来源

为确保估算模型的客观性与可验证性，本节将系统梳理并界定模型所需的全部核心变量，并明确其取值依据均来源于参考资料中的公开数据与行业标准。

首先，**运营里程**是决定车辆总运行里程的基础。根据中国城市轨道交通协会发布的《城市轨道交通2024年度统计和分析报告》，截至2024年底，全国共有41个城市拥有地铁，地铁运营里程为9306.09公里，占城轨交通总里程的76.53%[^9]。对于具体城市的里程排名，交通运输部发布的《2025年城市轨道交通运营数据速报》提供了截至2025年末的详细数据：北京以909公里位居第一，上海881.5公里第二，广州772.6公里第三，成都719.5公里第四，深圳634.5公里第五[^10]。武汉、南京、杭州、重庆、西安等城市也位列前十，里程均在400公里以上[^10][^11]。这些数据构成了城市级用量分析的基础。

其次，**车辆保有量**直接决定了碳滑板的装车基数。截至2024年底，全国城轨交通累计配属车辆已超过7.25万辆[^12]。一份截至2024年6月30日的统计显示，车辆保有量排名前十的城市依次为北京、上海、深圳、成都、武汉、南京、广州、西安、重庆和杭州，这十个城市的车辆合计高达7776列（约47537辆）[^13]。更具体的数据显示，北京和上海配置的列车均超过1000列（分别为1228列和1172列）[^9]。**车辆密度**（每公里运营里程对应的车辆数）是连接里程与车辆数的关键中间变量，可通过各城市的车辆保有量与运营里程计算得出。

第三，**单车碳滑板配置数**取决于受电弓的设计。根据国家标准《GB/T 38955-2020 城市轨道交通车辆用炭滑板》，该标准适用于地铁、轻轨等车辆的受电弓炭滑板产品[^6]。行业实践中，常见的单臂受电弓通常配置2根碳滑板，而部分车型或特定设计的受电弓可能配置4根（如采用双碳滑板条结构）。为进行保守与乐观的情景分析，模型将考虑这两种典型配置。

第四，**磨损率与更换标准**是计算消耗周期的核心。碳滑板的磨损分为机械磨损和电气磨损[^14]。在正常工况下，其磨损率相对稳定。例如，第一章分析的南京地铁案例中，碳滑板平均每万公里消耗约1.75mm。然而，**异常磨耗**是导致用量剧烈波动的关键风险。国内多个地铁线路曾出现弓网异常磨耗现象，导致碳滑板磨耗率几十倍增长，严重时车辆仅运行5000公里即需更换，寿命仅为正常情况（约10万公里）的二十分之一[^15][^14]。异常磨耗的原因包括大截面接触线的“锋刃”切削、新碳滑板更换后的磨合期拉弧、冬季温度骤降等[^15]。关于更换标准，动车组的通用要求是碳滑板磨耗后高度小于5mm时必须更换[^16]。地铁领域的具体规程可能略有不同，例如某“一号线”规定碳滑板厚度不低于磨耗到限标识或≥26.5mm，否则更换[^17]。模型将采用业界广泛认可的“剩余厚度小于5mm”作为更换阈值，并结合碳滑板的典型初始厚度（如32mm左右）来计算单根碳滑板的可磨耗厚度。

为清晰展示，现将模型核心变量、定义及数据来源汇总如下表：

| 变量类别 | 变量名称 | 定义/说明 | 主要数据来源与取值依据 |
| :--- | :--- | :--- | :--- |
| **基础规模变量** | 全国地铁运营里程 | 截至2024年底，41个城市地铁线路总长度 | 9306.09公里[^9] |
| | 主要城市运营里程 | 截至2025年末，各城市轨道交通运营里程 | 北京909km，上海881.5km，广州772.6km等[^10] |
| | 全国车辆保有量 | 截至2024年底，城轨交通配属车辆总数 | >7.25万辆[^12] |
| | 主要城市车辆保有量 | 截至2024年中，前十城市运营车辆列数 | 北京1228列，上海1172列等[^13][^9] |
| **技术配置变量** | 单车碳滑板配置数 (N_per_train) | 每列车受电弓标配的碳滑板根数 | 行业惯例：通常为2根或4根 |
| | 碳滑板初始厚度 (H_initial) | 新碳滑板的厚度 | 基于产品规格，假设为32mm |
| | 碳滑板更换阈值 (L_standard) | 必须更换时的最小剩余厚度 | 行业通用标准：<5mm[^16] |
| **消耗性能变量** | 正常磨损率 (W_rate_normal) | 良好弓网关系下的年均磨损厚度 | 参考案例：约1.75mm/万公里 |
| | 异常磨损率 (W_rate_high) | 发生弓网异常时的极高磨损率 | 参考资料：可达正常值的数十倍[^15][^14] |
| | 理论使用寿命 (T_life) | 基于磨损率与可磨耗厚度计算出的运行里程 | T_life = (H_initial - L_standard) / W_rate * 10000 |

### 2.2 模型构建、全国总量测算与城市级用量分析

基于上一节界定的变量，本节构建核心估算模型。模型的基本逻辑是：碳滑板的年度消耗量等于所有运营车辆在一年内因磨损而需要更换的总根数。

**1. 模型构建**
单根碳滑板的可磨耗厚度为：ΔH = H_initial - L_standard。假设H_initial = 32mm，L_standard = 5mm，则ΔH = 27mm。
基于磨损率（W_rate，单位：mm/万公里），单根碳滑板的理论使用寿命（公里）为：T_life_theory = (ΔH / W_rate) * 10000。
每列车年均碳滑板消耗根数为：C_train = (M_year / T_life_theory) * N_per_train。其中，M_year为单车年均运营里程，是一个需要通过行业数据估算的变量。一个合理的估算方法是：假设车辆全年无休高强度运营，每日运营约18小时，平均旅行速度约35km/h（参考全国地铁平均时速36.85公里[^9]），则单车日均运营里程约630公里，年运营里程（M_year）约23万公里。这是一个偏上限的估计，实际中车辆有检修、调配等因素，年均里程会低一些。为简化计算并保持一致性，后续测算将采用一个更贴近行业平均水平的假设值，例如M_year = 15万公里/列·年。
因此，一个城市的年度碳滑板总用量（根）估算公式为：Q_city = N_city * (M_year / T_life_theory) * N_per_train。
全国总用量则为各城市用量之和，或利用全国总量数据进行宏观估算。

**2. 全国总量测算**
采用宏观估算方法，利用全国地铁总里程和车辆总数进行推算。
已知：全国地铁里程L_national = 9306.09公里[^9]，车辆总数V_national > 7.25万辆（按7.3万辆计）。可估算全国平均车辆密度D_national = V_national / L_national ≈ 7.84辆/公里。
假设单车年均里程M_year = 15万公里，单车配置N_per_train = 3根（取2与4的中间值）。
设定两种磨损情景：
*   **情景A（正常磨损）**：取磨损率W_rate = 2.0 mm/万公里（略高于南京地铁案例，考虑全国平均水平）。则T_life_theory = (27 / 2.0) * 10000 = 13.5万公里。
*   **情景B（较高磨损/考虑部分异常）**：取磨损率W_rate = 4.0 mm/万公里。则T_life_theory = (27 / 4.0) * 10000 = 6.75万公里。

则全国年度碳滑板总用量估算为：
Q_national = L_national * D_national * (M_year / T_life_theory) * (N_per_train / 每列车车辆数换算) 。
更直接地，利用车辆总数计算：Q_national = V_national * (M_year / T_life_theory) * (N_per_train / 每列车平均编组辆数)。假设平均编组为6辆/列，则车辆数7.3万辆对应约12167列车。
计算可得：
- 情景A（正常）：Q_national_A ≈ 12167列 * (15 / 13.5) * 3根 ≈ **40,557根/年**。
- 情景B（较高）：Q_national_B ≈ 12167列 * (15 / 6.75) * 3根 ≈ **81,113根/年**。

**因此，基于模型测算，中国地铁碳滑板年度总用量估计在约4万根至8万根之间。** 这提供了一个基于当前运营规模和平均磨损水平的用量区间。实际用量可能因异常磨耗的集中发生而接近甚至短期超过区间上限。

**3. 城市级用量分析**
聚焦运营里程与车辆保有量均位居前列的“轨交第一梯队”城市——“京沪穗深蓉”[^10]。利用这些城市的具体数据进行测算，能更清晰地揭示用量规模与城市能级的关系。

### 2.3 运营强度影响分析与模型不确定性讨论

碳滑板的实际消耗不仅取决于“跑了多远”，更与“怎么跑的”密切相关，即运营强度。同时，模型估算存在固有的不确定性，需要明确其边界。

**1. 运营强度影响分析**
“客运强度”（万人次/公里·日）是衡量地铁线网繁忙程度与运营效率的核心指标[^9][^18]。高客运强度意味着单位里程承载的客流量大，通常对应更密的列车发车间隔、更频繁的启停和加减速。这些都会加剧碳滑板的机械摩擦和电气拉弧，导致磨损率上升，缩短更换周期，从而增加年度用量。
根据2024年数据，地铁客运强度最高的五个城市是：深圳（1.52万）、广州（1.51万）、长沙（1.32万）、上海（1.29万）和北京（1.28万）[^9]。将这些城市与2.2节的用量估算结合分析可以发现，深圳和广州的客运强度断层领先，其单位车辆或单位里程对应的碳滑板实际消耗速率很可能高于模型采用的全国平均磨损率假设。这意味着，对于这些超高强度运营的城市，其实际年度用量可能更接近甚至超过模型估算区间的上限。反之，客运强度较低的城市（如南通、绍兴，仅在2000人次左右[^9]），其碳滑板磨损更慢，实际用量可能低于基于里程的线性预测。**因此，在应用模型时，对高客运强度城市应酌情调高磨损率参数，以更贴近实际。**

**2. 模型不确定性讨论**
本估算模型是一个基于多变量假设的理论工具，其准确性受限于以下主要不确定性来源：
*   **磨损率参数的巨大波动性**：这是最大的不确定性来源。参考资料[^15][^14]详细分析的弓网异常磨耗表明，在特定线路、特定时段（如新线磨合、冬季），磨损率可能发生数量级的跃升，从正常的每万公里几毫米激增至几十毫米。模型中的区间估计虽然尝试涵盖这种可能，但无法预测异常事件发生的具体时间、地点和规模。
*   **数据口径与时效性差异**：不同来源的统计数据存在细微差别。例如，交通运输部《速报》的里程排名截至2025年末[^10][^19]，而中国城市轨道交通协会的《年度报告》数据截至2024年底[^9]。车辆保有量的统计时点（2024年底与2024年中）和具体列数也有不同出处。模型在整合时以最新或最权威数据为准，但这种差异是客观存在的。
*   **维护策略与实践的多样性**：各地铁公司的运营维护策略直接影响碳滑板的实际寿命。例如，是否定期对接触线侧棱和碳滑板进行预防性打磨[^15]，是否在碳滑板未到极限厚度前就进行预防性更换，以及采用不同品牌、材质的碳滑板其耐磨性本身就有差异。这些因素都使得“一刀切”的磨损率参数无法精确反映每个城市的实际情况。
*   **供应链与库存因素**：模型测算的是理论消耗量，而非实际采购量。实际采购行为受到库存水平、采购周期、批量订单策略以及突发性批量更换（如应对异常磨耗）的强烈影响，可能导致年度采购量在理论消耗量基础上大幅波动。

**结论**：本章构建的碳滑板年度用量估算模型，基于公开的行业统计数据和技术标准，提供了全国及重点城市需求量的量化参考区间（约4-8万根/年）。分析表明，**超大规模城市（京沪穗深蓉等）是需求的主体，且其高客运强度可能导致实际消耗偏向区间上限**。然而，模型本质上是一个基于平均化假设的工具，其最大不确定性源于弓网异常磨耗带来的磨损率剧烈变化。因此，该模型更适用于理解市场总体规模、结构及主要驱动因素，为行业分析和战略规划提供依据，而非用于精确的短期采购预测。

## 3 主要供应商竞争格局与市场份额分析

本章节在明确碳滑板核心价值与消耗规律的基础上，聚焦于中国地铁碳滑板市场的竞争主体。通过系统梳理2024-2025年各地铁公司的公开招标与采购结果，并结合行业研究报告的关键洞察，本章旨在绘制清晰的市场参与者图谱，基于实证数据分析各供应商的竞争力，并对其市场份额进行估算与排序，从而揭示当前市场的集中度、竞争态势及未来格局的潜在驱动因素。

### 3.1 市场主要参与者图谱：制造商、代理商与集成商

中国地铁碳滑板市场参与者多元，角色清晰，主要可分为核心制造商、国际品牌及在华实体、代理商/贸易商以及主机厂关联企业四大类。以下结合参考资料中的具体案例，对各类型参与者进行梳理。

**1. 国内核心制造商**
国内制造商通过持续的技术研发与产业化，已成为市场的中坚力量，并在国产替代进程中扮演关键角色。
*   **苏州东南佳新材料股份有限公司**：是国内碳滑板领域的领军企业。资料显示，其是国内**首家拥有独立自主知识产权的碳滑板生产厂商**，拥有发明、实用新型专利20多项[^20]。该公司市场表现活跃，在2025年多个地铁招标项目中位列前茅，例如：中标**杭州地铁5号线2025-2027年浸金属碳滑板采购项目**，金额155.9067万元[^21][^22]；在**南昌轨道交通运营分公司2025年度检修专业碳滑板采购项目**中位列第一中标候选人[^23][^4][^24]；在**洛阳市轨道交通2025年度带ADD气道碳滑板采购项目**中位列第三成交候选人[^25][^26]。其产品已成功应用于“和谐号”及“复兴号”动车组，并为上海、广州、深圳、郑州、武汉等全国20多个地铁公司配套[^20][^27]。公司总部位于江苏，并在湖北仙桃设有重要生产基地，被认定为国家级专精特新“小巨人”企业[^27]。
*   **北京万高众业科技股份有限公司**：是市场中的另一主要竞争者，频繁出现在各地招标的中标候选人名单中。在**洛阳市轨道交通2025年度带ADD气道碳滑板采购项目**中，其为第二成交候选人，响应报价723,652.00元[^25][^26]；在**南昌轨道交通运营分公司2025年度检修专业碳滑板采购项目**中为第二中标候选人[^23][^4]；在**运营集团2025年车辆专业碳滑板组件采购项目（第二次）**中为第二中标候选人，投标报价89.7585万元[^28][^26]。该公司注重技术创新，已取得名为“碳滑板及受电弓”的实用新型专利，旨在延长碳条使用寿命[^4]。根据全球市场研究报告，其被列为全球主要受电弓碳滑板生产商之一[^29]。
*   **辽宁红德新材料科技有限公司**：作为中国国家铁路集团有限公司的定点生产厂家，以及中车青岛四方、中车长客、中车唐山等多家主机厂的合格供应商，其资质深厚[^30][^31][^32]。公司产品包括电刷、碳滑板（含除冰碳滑板）等，其官网展示的浸金属碳滑板产品具有粘接电阻低、导电导热性好等特点，并提供了具体的理化指标数据[^33]。其市场应用已覆盖哈尔滨、沈阳、北京、太原等多个铁路局[^32]。
*   **大同新成新材料股份有限公司**：专注于特种石墨生产，是**全国炭素行业《细颗粒高密度特种石墨产品》国家标准的主持制定单位**，拥有专利637项，是国内特炭生产的领军企业[^24]。公司通过自主研发，攻克了浸金属碳滑板技术，成为国内首家拥有独立自主知识产权且全部零部件国产化的高铁碳滑板生产企业之一，其“高铁动车组碳滑板”产品成功入选首批“山西精品”[^24]。

**2. 国际品牌及在华实体**
国际品牌凭借长期的技术积累，在高端市场和部分早期项目中保有影响力。
*   **崇德集团（Schunk）**：德国品牌，技术实力雄厚。早期案例显示，南京地铁一号线、二号线曾采用其整体式碳滑板[^1]。更重要的信息是，崇德碳技术（苏州）有限公司宣称，其碳滑板**占据中国高铁最大市场份额**，并且是**全球地铁碳滑板市场份额最大的供应商**[^34][^35][^22]。其改进型浸金属碳滑板及焊接结合粘接的工艺被认为是行业领先技术[^34]。该公司已为国内近40条城轨线路供应受电弓[^34]。
*   **摩根先进材料（Morgan Advanced Materials）**：在全球市场研究报告中，与Mersen、Schunk等一同被列为全球主要受电弓碳滑板生产商[^29]。南京地铁运营有限责任公司在2024年曾就“摩根碳滑板”进行专项采购[^36]，表明其在特定线路或车型上作为指定品牌存在。
*   **法中轨道交通运输设备（上海）有限公司**：在**广州地铁集团有限公司2024年受电弓碳滑板备件采购项目**中，因其货物或服务使用不可替代的专利、专有技术，成为单一来源谈判的成交单位[^37]，显示了国际品牌在特定技术领域的独占性。

**3. 代理商/贸易商**
此类企业作为产品流通的渠道，在地方性采购中扮演重要角色。
*   **天津市众远航自动化科技有限公司**：在天津轨道交通的采购中表现突出，是**运营集团2024年车辆专业6、9号线碳滑板及组件采购项目**的中标人（101.6688万元）[^38]，同时也是**2025年车辆专业碳滑板组件采购项目（第二次）**的第一中标候选人（89.2000万元）[^28][^26]。
*   **天津市雯硕科技有限公司、上海鼎襄机电设备有限公司**等也频繁出现在各地招标的候选人名单中[^28][^25][^23]，构成了市场供应网络的补充部分。

**4. 主机厂关联企业**
依托系统集成优势，直接参与部件供应。
*   **乌鲁木齐中车轨道交通装备有限公司**：在**长沙市轨道交通1、2、3、4、5号线运营期2025年碳滑板采购项目**中，被确定为第一成交候选人[^39][^40]。
*   **株洲国创轨道科技有限公司**：因其自主研发的**碳纤维碳滑板**被认定为行业内无同类产品，在**郑州市轨道交通2025-2026年碳滑板异常磨耗整治合作及采购项目**中，被采购方采用直接委托方式确定为供应商[^22][^24]。

从采购单位分布看，供应商已形成了全国性的服务网络。苏州东南佳、北京万高众业等头部企业的客户遍及华东（杭州、南昌）、华中（洛阳、郑州）、华北（天津）等多个区域的核心城市。同时，市场也存在明显的区域特性，例如天津市场由本地贸易商主导，而广州、成都等地则有较强的本地或关联企业参与。这种格局表明，全国性品牌与区域性供应商并存，市场竞争呈现**全国渗透与区域深耕相结合**的特点。

### 3.3 市场份额估算、竞争态势与未来格局展望

综合市场参与者图谱与招标实证分析，可以对当前市场份额、竞争态势进行估算与总结，并展望未来格局的演变。

**1. 市场份额估算与市场集中度**
由于无法获取完整的市场销售数据，市场份额估算主要基于行业报告结论与招标市场表现的交叉验证。

基于以上信息，可以对中国地铁碳滑板市场的竞争梯队进行定性描述：
*   **第一梯队（国际技术领导者与高端市场保有者）**：以**崇德（Schunk）**为代表，凭借深厚的技术积淀和品牌影响力，在部分高标准要求线路、早期项目或特定技术方案中占据重要位置。
*   **第二梯队（市场主导与国产替代主力）**：以**苏州东南佳新材料**和**北京万高众业科技**为核心，加上**辽宁红德**、**大同新成新材料**等具备自主知识产权的国内头部制造商。**这个梯队是当前地铁碳滑板采购市场中最活跃、份额占比最大的群体**，尤其在后市场（运营维修）替换领域优势明显。
*   **第三梯队（区域代理商、贸易商及特色供应商）**：包括各地招标中出现的贸易公司，以及像**株洲国创**这样拥有独家新技术的企业，它们填补细分市场，使竞争格局更加多元化。

**2. 当前竞争态势总结**
当前中国地铁碳滑板市场竞争态势呈现以下特点：
*   **国产替代成效显著**：以东南佳、万高众业为代表的国内企业，通过自主研发，产品性能已能满足地铁运营要求，并在成本、本地化服务和快速响应上具备优势，已成为市场采购的主力选择。
*   **竞争要素多元化**：竞争已从单一价格竞争，转向**技术、质量、服务、品牌、供应链稳定性**等综合能力的比拼。能否提供解决特定问题（如异常磨耗）的方案成为新的竞争力体现。
*   **市场集中度较高**：无论是行业数据还是招标结果，都指向市场向头部企业集中，但进入者依然存在机会，尤其是在新技术、新材料或特定区域市场。

**3. 未来格局驱动因素与展望**
未来市场竞争格局将受到以下几大关键因素的深刻影响：
*   **技术持续迭代与材料创新**：下一代复合材料（如碳纳米管增强型）的产业化将是决定性的技术变量[^25][^45]。率先实现技术突破并量产的企业将获得巨大竞争优势。同时，针对**异常磨耗**的专项解决方案（如郑州地铁采购的新型碳纤维碳滑板）的需求将持续增长，催生新的市场机会[^22]。
*   **供应链安全与成本控制**：行业报告尖锐指出，中国高纯铝（电子级）与特种石墨原料**进口依赖度分别高达68.3%和53.6%**，导致原材料成本波动剧烈，直接影响行业毛利率和企业盈利能力[^25][^45]。构建稳定、有韧性的供应链将成为企业的核心战略能力。
*   **国产产品性能的全面追赶**：尽管国产滑板在高速工况下的磨损率（0.85mm/万弓架次）仍与德国（0.38mm）、日本（0.42mm）产品有差距，但通过定制化改进（如高原型），其服役寿命-成本比（SLCR）已显著提升，在部分场景下逼近日系水平[^25]。**性能差距的持续缩小将不断侵蚀国际品牌在高端市场的传统优势**。
*   **政策与标准导向**：“双碳”目标下的环保要求、以及对装备国产化率的鼓励政策，将持续为国内优势企业创造有利环境。

**结论**：中国地铁碳滑板市场是一个由技术驱动、国产化主导、且集中度较高的专业市场。以苏州东南佳、北京万高众业为代表的国内头部制造商，凭借技术突破、市场快速响应和综合成本优势，已成为市场的中坚力量，并在持续扩大份额。未来，竞争将更加聚焦于新材料研发、供应链韧性构建以及针对运营痛点的深度解决方案提供能力上，技术领先和生态整合能力强的企业将在格局演变中占据主动。

## 4 行业发展趋势综合研判

本章基于前述章节对市场规模、竞争格局的量化分析，结合新材料、智能检测、供应链韧性及政策导向的最新信息，对中国地铁碳滑板行业的未来发展方向进行系统性研判。行业正处在由规模扩张向高质量发展转型的关键节点，其演进路径将深刻受到技术突破、运维模式革新、供应链安全以及宏观政策等多重因素的共同塑造。本章旨在从技术、市场、政策、竞争四个维度，综合揭示行业面临的机遇、挑战与潜在演变路径。

### 4.1 材料技术迭代：从性能追赶到定制化创新

碳滑板材料技术的演进正沿着高性能化和定制化两个主要方向加速推进，其核心驱动力是满足高速重载运营、降低全生命周期成本以及适应复杂运行环境的需求。当前，国产材料在关键性能指标上与国际顶尖水平仍存在可量化的差距，这构成了技术追赶的明确起点，同时也催生了通过创新路径实现超越的机遇。

**首先，性能基准揭示了明确的追赶空间。** 根据一份针对铝包浸金属碳滑板的深度市场研究报告，在350公里/小时的高速工况下，德国Schunk与日本东洋炭素产品的磨损率分别低至0.38mm/万弓架次和0.42mm/万弓架次，而国产主流产品的磨损率约为0.85mm/万弓架次[^45]。在表征电流传输稳定性的动态接触电阻标准差指标上，国产滑板为1.43mΩ，明显高于德日产品的0.82-0.95mΩ范围[^45]。这些差距主要源于国产石墨基体的石墨化度偏低、铝包覆层均匀性不足以及界面结合强度较弱等基础材料与工艺问题[^45]。**这一性能鸿沟直接影响了产品在高速场景下的服役寿命与可靠性，是国产高端化必须攻克的核心技术壁垒。**

**其次，以合金碳复合材料为代表的下一代材料展现出巨大的替代潜力。** 合金碳滑板以轻质合金为基体、碳纤维复合材料为增强相，通过材料复合工艺有机整合了金属的导电性与碳材料的耐磨性[^25][^26]。实验室数据表明，其典型制品的单位磨耗量较传统铜基滑板降低40%以上，同时保持85%以上的导电效率；抗冲击强度达到纯碳材料的2.3倍，能有效避免高速过弯时的断裂事故[^25][^26][^45]。尽管其原材料成本较常规产品高出25%-40%，但全生命周期成本模型显示其经济优势显著：某重载铁路实测数据表明，更换周期从传统滑板的3个月延长至9个月，按十年运营周期计算，综合成本可降低18%以上[^25][^26]。**这预示着合金碳复合材料凭借其综合性能优势，有望成为未来高速、重载地铁线路的主流选择，市场增长潜力巨大（年复合增长率达17%）[^25][^45]。**

**再者，先进制造工艺与定制化创新成为提升竞争力的关键路径。** 一方面，3D打印、真空压力浸铜等先进工艺正在应用。例如，采用3D打印技术结合碳纤维布、铜纳米颗粒等原料，并通过1300-1400℃真空浸铜后处理的滑板，其耐磨性较传统工艺提升42%，导电率提高28%[^46]。增材制造还使复杂结构件成品率提升至95%，微波固化技术可缩短生产周期40%[^25][^45]。另一方面，针对特定恶劣环境的定制化产品开发成效显著。报告指出，国产滑板通过在西部低密度、高海拔线路推出“高原型”定制产品（如铝层增厚至0.6mm、引入Al-Si合金），其服役寿命-成本比（SLCR）从0.89大幅提升至1.21，已逼近日系产品水平[^45]。这种从通用产品向“区域适配型”解决方案的转变，体现了技术创新的深度与灵活性。

然而，技术产业化仍面临三重障碍：高端碳纤维产能受国际供应链制约；复合界面稳定性需突破百万次摩擦的耐久极限；行业检测标准滞后导致新产品认证周期过长[^25][^45]。**综合来看，材料技术迭代正沿着“对标追赶—工艺革新—场景定制”的路径演进，下一代复合材料的产业化进程与供应链自主化能力，将决定未来市场竞争的制高点。**

### 4.2 智能运维驱动：从“故障修”到“精准修”的范式变革

智能检测与预测性维护技术的快速发展，正在深刻重塑碳滑板的运维管理模式、行业成本结构乃至核心竞争要素，推动行业从被动应对的“故障修”、定期执行的“预防修”向数据驱动的“精准修”全面转型。

**传统人工检测的痛点催生了智能化解决方案的迫切需求。** 传统人工登顶检测法耗时费力，测量精度受人为因素影响大，且无法实现列车不停靠检测，高空作业也存在安全隐患[^47]。更为关键的是，碳滑板的异常磨耗是地铁运营中的常见风险，据不完全统计，全国包含南京地铁在内的20多家地铁均出现过此类情况，极易引发次生的弓网病害，极大增加运营风险与成本[^48][^46]。因此，强化状态检测、实现早期预警成为行业刚需。

**多种智能检测技术已实现从实验室到工程现场的突破性应用。** 目前，市场上已涌现出多类先进的非接触式、自动化检测装置，其技术路径与性能各具特色：
*   **激光扫描与点云处理技术**：以天弓智测科技创新团队研发的系统为代表，通过激光测距仪沿碳滑板轨迹扫描，获取三维点云数据，可进行尺寸测量、缺陷检测和形状评估，快速生成可视化磨耗报告[^47]。
*   **便携式智能检测仪**：如南京地铁联合研发的便携式设备，采用轻量化镁铝合金设计，单次检测可在48-70秒内采集超过5000组数据，检测精度高达0.01毫米，并能自动匹配磨耗最低点与最高点，实现自动测量、计算与研判[^48][^41][^49]。
*   **超声波与多模态传感阵列**：例如“易受难弓”团队研发的系统，通过高频声波反射构建三维成像模型，结合AI算法识别毫米级裂纹和碳层剥离等隐患，检测效率较传统方式提升80%，误报率控制在0.5%以下[^48][^46]。
*   **集成化机器人检测系统**：铁建交运等企业将“门型检测机器人”、可移动检修机器人集成于自动化车辆检修库，配合高清摄像机、激光测距仪等传感器，实现列车经过时对碳滑板及车下部件的同步快速检测，将人工检测效率提升40%[^49][^38]。

为了更清晰地对比各类技术的特点，以下表格进行了归纳：

| 技术类型 | 核心原理/设备 | 关键性能指标 | 主要优势 | 应用案例/团队 |
| :--- | :--- | :--- | :--- | :--- |
| **激光扫描** | 激光测距仪、运动控制单元 | 三维点云数据、形状评估 | 非接触、高精度三维建模 | 天弓智测科技创新团队[^47] |
| **便携式设备** | 轻量化镁铝合金机身、集成传感器 | 检测时间≤70秒，精度0.01mm，数据>5000组/次 | 轻便、快速、自动化程度高 | 南京地铁运营公司[^48][^41] |
| **超声波传感** | 超声波多模态传感阵列、AI算法 | 识别毫米级缺陷，检测效率+80%，误报率<0.5% | 深层缺陷检测、高精度识别 | “易受难弓”团队[^48][^46] |
| **机器人集成** | 门型检测机器人、可移动检修臂、多传感器融合 | 实现不停车同步检测，人工效率提升40% | 自动化、高效率、集成化 | 铁建交运[^49][^38] |

**这一技术变革正在引发运维范式与商业模式的深刻重塑。** 首先，运维模式从“计划驱动”转向“数据驱动”。基于实时检测数据与AI预测模型，运营单位可以精准掌握每块碳滑板的剩余寿命，实现“按需更换”，从而大幅减少过度维护和意外故障，相关资料显示可帮助降低30%以上的维护成本[^48][^46]。其次，竞争要素从“单一产品”扩展到“整体解决方案”。供应商的竞争力不再仅取决于碳滑板本身的性能，更在于能否提供涵盖智能检测硬件、数据分析平台、寿命预测算法及专家诊断服务的全套运维方案。例如，一些检测系统已配套弓网异常磨耗病害的修复装置及专家库系统[^41][^49]。**最终，行业价值重心正从“制造”向“服务”延伸，智能化运维能力将成为区分市场领导者与跟随者的关键标志。**

### 4.3 供应链韧性挑战与国产化深化

碳滑板行业的上游供应链安全已成为制约产业稳健发展的战略性瓶颈。关键原材料的对外高度依赖，不仅导致成本剧烈波动，侵蚀企业盈利能力，更在复杂的国际经贸环境下构成了潜在的“断供”风险。推动供应链自主可控和国产化深化，已从发展选项升级为生存必需。

**核心原材料进口依赖度高是最大的风险敞口。** 市场研究报告明确指出，中国碳滑板生产所需的关键原材料——**高纯铝（电子级）与特种石墨的进口依赖度分别高达68.3%和53.6%**[^45]。这种结构性依赖使得国内企业严重暴露于国际原材料价格波动与供应链中断的风险之下。数据显示，2022年至2024年间，受此影响，行业毛利率从34.5%的高位被压缩至22.8%[^45]。**原材料成本已成为影响企业利润最敏感、最不可控的外部变量之一。**

**国产化替代在材料与工艺两端均面临艰巨挑战。** 在材料端，国产特种石墨在石墨化度、纯度、结构均匀性等指标上与进口高端产品存在差距，直接影响碳滑板的导电稳定性与耐磨性[^45]。在工艺端，例如国产热压浸渗工艺的良率与国际标杆企业相比仍有明显的量化差距[^45]。同时，欧美“近岸外包”的供应链回流趋势，可能进一步加剧关键材料获取的不确定性和成本压力[^45]。这些挑战意味着国产化替代不能简单照搬，必须通过技术创新实现性能与成本的平衡。

**构建韧性供应链需要多维度战略应对。** 面对挑战，行业与企业正在采取多种策略：
1.  **垂直整合与战略合作**：国内领先的制造商，如大同新成新材料股份有限公司，本身就是特种石墨生产的龙头企业，通过向上游延伸掌握了核心材料技术[^46]。其他企业则通过与国内矿企、科研院所建立战略联盟，共同研发替代材料，降低对单一进口源的依赖。
2.  **工艺创新与精益生产**：为了克服原材料性能的细微差距，企业投入研发更精密的制备工艺。例如，3D打印结合真空浸铜的工艺，通过对材料微观结构的精确控制，实现了产品性能（耐磨性、导电性）的显著提升，部分程度上弥补了原材料基础的不足[^46]。
3.  **绿色供应链与循环经济**：在“双碳”政策目标下，环保法规对高耗能生产环节的约束日益严格[^42][^37]。这倒逼企业优化生产工艺、降低能耗，并探索碳滑板基体合金（回收率可达92%）等材料的回收再利用，这不仅是环保要求，也构成了新的成本控制与供应链稳定途径[^25]。

**因此，供应链的竞争已超越传统的成本与交货期范畴，升级为涵盖原材料自主、工艺know-how、绿色可持续性在内的综合韧性体系竞争。** 能够在这一体系中建立优势的企业，将在未来的行业波动中占据更主动的位置。

### 4.4 政策与市场双轮驱动下的需求演变与格局展望

碳滑板行业的未来发展，将在国家宏观战略与下游市场动态的“双轮驱动”下，呈现出需求结构持续优化、竞争格局加速集中的鲜明趋势。把握政策导向与市场脉搏，是理解行业演进逻辑的关键。

**从需求侧看，增长动力正从“新建扩张”向“存量更新与升级”并重演变。**
1.  **增量基本盘依然稳固**：截至2025年末，全国城市轨道交通运营里程已达11710.3公里，且上海等超大城市已原则同意新一轮（2025-2035年）线网规划，预示着未来十年轨道交通网络将持续扩张，为碳滑板带来稳定的新增装车需求[^26][^50][^44]。
2.  **存量更新市场加速释放**：随着早期建设的地铁线路陆续进入设备更新周期，更新改造市场正驶入“快车道”[^37]。全国已有超过77条运营超过10年的城轨线路，其设施设备将陆续进入批量更换阶段[^37]。这一存量市场的规模巨大且需求持续性强。
3.  **需求内涵向高性能、绿色化升级**：无论是新建线路还是更新改造，在“双碳”战略和运营经济性双重驱动下，对碳滑板产品的需求不再满足于“能用”，而是追求“好用、耐用、省心”。**长更换周期、低全生命周期成本、高可靠性、环保可回收**成为核心采购考量[^25][^42][^37]。这直接利好于掌握了合金碳复合材料等先进技术的供应商。

**基于技术、供应链与政策的综合影响，未来市场竞争格局将呈现以下演变趋势：**
*   **市场集中度进一步提升**：技术门槛的提高（新材料、智能化）、供应链韧性要求以及规模效应，将使得资源向头部企业聚集。全球市场已呈现由少数跨国集团主导的“金字塔结构”[^25][^45]。在中国市场，以苏州东南佳、北京万高众业为代表的国内头部制造商，凭借技术突破、成本优势和快速市场响应，已占据主导地位，预计这一趋势将延续并强化。
*   **竞争壁垒从“制造”迈向“科技与服务”**：未来的行业领导者，必然是能够提供“高性能材料+智能监测+数据服务”一体化解决方案的科技型企业。单纯的生产制造能力或价格优势，难以构成长期护城河。企业需要同时具备材料研发、精密工艺、传感器与算法开发、大数据平台运营等多学科融合创新能力。
*   **国产化替代从“中低端”向“全领域”纵深推进**：国产碳滑板不仅在运营维修市场占据绝对份额，随着性能持续提升（如SLCR优化）和解决特定工况能力的增强，正在向新建线路、高速重载等高端应用场景渗透，与国际品牌展开正面竞争[^45]。
*   **出海机遇与挑战并存**：依托国内庞大的应用场景锤炼出的产品与解决方案，以及“一带一路”倡议的推动，中国碳滑板企业具备出海潜力。然而，这需要应对国际标准认证、本地化服务、品牌建设以及与国际巨头的直接较量等复杂挑战。

**综上所述，中国地铁碳滑板行业正步入一个以技术驱动、服务增值和供应链安全为核心特征的新发展阶段。** 拥有核心技术、具备一体化解决方案交付能力、并构建了稳健供应链体系的头部企业，将在格局演变中持续扩大优势。而对于整个行业而言，能否在材料科学上实现根本性突破，并成功将智能化运维模式规模化落地，将决定其能否在全球轨道交通产业链中占据更重要的价值地位。
